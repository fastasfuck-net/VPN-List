# VPN & Proxy Blocklist

Automatically updated lists of VPN servers, proxy servers, and Tor exit nodes used for detection and blocking.

---

## Data Sources

This repository aggregates data from the following open-source projects.

| Source | Repository | License |
|--------|-----------|---------|
| Free Proxy List | [proxifly/free-proxy-list](https://github.com/proxifly/free-proxy-list) | [GPL-3.0 license](https://github.com/proxifly/free-proxy-list/blob/main/LICENSE) |
| PROXY-List | [TheSpeedX/PROXY-List](https://github.com/TheSpeedX/PROXY-List) | No license – credits requested by author |
| proxy-list | [monosans/proxy-list](https://github.com/monosans/proxy-list) | [MIT](https://github.com/monosans/proxy-list/blob/main/LICENSE) |
| VPN Gate | [vpngate.net](https://www.vpngate.net) | Public API |
| Tor Exit Nodes | [torproject.org](https://www.torproject.org) | [Tor License](https://gitweb.torproject.org/tor.git/plain/LICENSE) |
| VPN-list by X4BNet | [github.com/X4BNet/lists_vpn](https://github.com/X4BNet/lists_vpn) | [MIT License](https://github.com/X4BNet/lists_vpn/?tab=readme-ov-file#license) |
